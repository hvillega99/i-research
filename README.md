
# i-research

Repositorio de investigadores de ESPOL

## Tech Stack

**Frontend:** EJS, Bootstrap

**Backend:** Node, Express, Redis


## Requerimientos
- Node 14.15 o superior
## Despliegue

- instalar dependencias

```bash
  npm i
```
- configurar cas (cas.config.json) y redis (redis.config.json)

- levantar servidor

```bash
  npm start
```
